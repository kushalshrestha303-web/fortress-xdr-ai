# Integration Packs

Integration packs follow an XSOAR-like content structure while staying vendor-neutral:

- `Integrations`: API clients and commands.
- `Playbooks`: task graphs for incident response.
- `Scripts`: reusable automation.
- `Dashboards`: SOC widgets and metrics.
- `Layouts`: incident and evidence views.

All packs must be schema-validated, tested, signed, and tenant-scoped before deployment.
